About
============
If you want to get excel reports of sales order/quotation. So here we build a module that can help to print the excel report of the sales order/quotation. You can get an excel report separate sheet of each order also. Cheers!

Installation
============
1) Copy module files to addon folder.
2) Restart odoo service (sudo service odoo-server restart).
3) Go to your odoo instance and open apps (make sure to activate debug mode).
4) click on update app list. 
5) search module name and hit install button.
