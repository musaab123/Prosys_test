Product Selection
To print labels by the wizard, go to the "Purchase" - "Orders" - "Purchase Orders" menu and select one or several purchase orders. Then click on the "Custom Product Labels" in the "Print" menu.
By using our additional modules, you will also be able to select products for printing from: Stock Pickings, Product Packaging, Manufacturing Orders.
Label Settings
Label Settings
In the print wizard form:
  1. Select a Label Template.
    2. Templates can be in a variety of designs, sizes, and paper formats. Look at our other related modules.
  2. Change the number of labels that will be printed (if necessary).
    3. You can enter a quantity value or use / buttons.
  3. If you need to set a specific quantity for each label, enter the value and click on Set quantity.
  4. After changing the label quantities, you can restore the initial values by clicking the button .
  5. To turn off printing for a specific label, deactivate the "Print" option.
Label Options
You can also set some label options on the tab Options:
  • Language - to specify the language to translate label fields.
  • Human readable barcode - to print a barcode digit code on the labels.
  • Border - to set the label border width.
    • Set to 0 to print labels without border.
Label Printing
Finally, you can preview the labels before printing, by clicking on the PREVIEW button, or print them with the PRINT
